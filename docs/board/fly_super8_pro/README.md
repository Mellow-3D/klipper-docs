# 配置信息

* MCU: STM32H743
* 支持CANFD
* 板载ESP32WIFI模块，RRF固件传输速率最大2.43Mbytes/S
* 8个高压驱动位
* PCB采用2OZ铜厚，大电流支持
* 10个可控风扇，可插拔MOS
* 6个ADC接口
* 8个IO接口
* 2个PMW接口
* 1个高压IN口
* 5个加热棒接口

?> Super8 Pro除了`主控芯片`与Super8不相同之外，其余接线和引脚配置均可通用

Super8相关教程：[Fly-Super8](/board/fly_super8/ "点击即可跳转")
