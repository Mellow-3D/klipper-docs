# 配置信息

* MCU: STM32H723
* 支持CANFD
* 可扩展ESP32WIFI模块，RRF固件传输速率最大2.43Mbytes/S
* 5个高压驱动位
* PCB采用2OZ铜厚，大电流支持
* 4个可控风扇，可插拔MOS
* 3个ADC接口
* 6个IO接口
* 1个高压IN口

----

?> 重要

* 非FLY上位机请按[CAN使用](/advanced/can_rpi.md)文档配置好CAN
* 使用CanBoot请查看[CanBoot使用](/advanced/canboot.md)
