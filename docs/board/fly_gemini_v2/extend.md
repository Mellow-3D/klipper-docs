# 2. RPFMEX-4轴扩展板

为了满足客户对Gemini的多轴需求，我们做了4轴扩展板，让原来的Gemini 4轴变成8轴，满足更多3D打印机的需求。 此扩展板不能单独用作 3D 打印机主板，因为它不包含加热和温度测量模块。
