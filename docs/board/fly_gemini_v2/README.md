# 配置信息

* SOC: 全志H5
* MCU: STM32F405
* RAM: 1GB
* ROM: 支持最大128GB SDCARD
* 外设: SPI\*2, UART\*2, USB\*4, Micro HDMI\*1, ETH\*1, CAN
* 接口：板载M.2接口，可拓展wifi，emmc等。(私有协议，切勿连接非FLY-Gemini专用的M.2设备)
* 驱动: 高压驱动位*4

----

* 非FLY上位机请按[CAN使用](/advanced/can_rpi.md)文档配置好CAN
