# 产品简介

Fly-Pi-lite是一款入门级Klipper上位机

**特点**

* CPU：高性能全志T113-S3芯片，双核A7
* RAM：128M+（1G SWAP）
* ROM：支持最大128GB SDCARD
* 外设：Spi x2，UART x2，USB x2
* 接口：预留FPC
* 为Klipper定制的系统，无需安装系统等复杂的操作，小白更容易上手
